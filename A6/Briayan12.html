Software libre

Escritorio

7-zip

7-zip es un comprensor/descompresor de archivos que maneja los formatos ZIP, RAR, CAB, GZIP, BZIP2, y TAR, así como un formato propio. 7z.

Audacity

Audacity es un editor de audio. Permite reproducir, grabar, editar, mezclar, aplicar efectos, etc.

GIMP

GIMP (GNU Image Manipulation Program) es un programa de retoque fotográfico y comoposición y edición de imaágenes.

Inkscape

Inkscape es un editor de gráficos vectoriales que utiliza el formato SVG y que también importa formatos como Postscript, EPS, JPEG, PNG Y TIFF y exporta en PNG y varios formatos vectoriales.

LibreOffice

LibreOffice es una suite ofimática completa compuesta de procesador de texto, hoja de cálculo, presentaciones, programa de dibujo, base de datos y editor de ecuaciones.

Internet

Filezilla

FileZilla es un cliente de FTP.

Firefox

Firefox es un navegador web altamente personalizable y ampliable mediante los cientos de extensiones y temas disponibles.

HTTrack

HTTrack permite descargar sitios webs completos para su consulta offine.

Pidgin

Pidgin es un cliente de mensajería instantána, compatible con las redes AIM (protocolos Oscar y TOC), ICQ, MSN Messenger, Yahoo, IRC, Jabbe, Gadu-Gadu, Zephyr y otras.

Thunderbird

Thunderbird es un cliente de correo de los creadores de Firefox. Incorpora multitud de características avanzadas y, como Firefox, puede personalizarse mediante temas y ampliar sus posibilidades mediante extensiones.
